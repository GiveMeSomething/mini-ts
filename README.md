# Mini TS

## Description

Setup a quick playground with TypeScript for testing / POC

## Installation

- Install dependencies

```
npm install
```

- Run the code in main (dev mode)

```
npm run dev
```

## Notes

- Build and run the code

```
# Build the server to dist folder
npm run build

# Run the server
npm start
```

- Format code base

```
npm run format
```

- Lint code base

```
npm run lint
```
